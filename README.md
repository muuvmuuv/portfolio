## Development

Developing this project is easy, simply run this in your shell:

```
pnpm start
```

## Linting

```
pnpm run lint
```

## Deployment

I'm using [Vercel](https://vercel.com/) to deploy my site.

## Release

```
pnpm run release
```

Commits follow [conventional commits](https://www.conventionalcommits.org/) flow and must
look like:

```yml
<type>: <description>

[optional body]

[optional footer]
```
